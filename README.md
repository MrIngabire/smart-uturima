# Smart Uturima System
**Internship Capstone Project - IDA Technology**

This project is a Full-Stack Urban Gardening Management System developed during a 12-week internship at IDA Technology.

### Technical Stack
- **Backend:** Python / Django REST Framework
- **Frontend:** React.js / Axios
- **Automation:** Custom Python logic for sensor-threshold tasks.

### Key Features
1. **Garden Tracking:** Managing community kitchen gardens.
2. **AI Automation:** Automatic task generation for low moisture levels.
3. **Data Visualization:** Real-time analytics dashboard for stakeholders.
